# 🤖 SocialSelf – AI-Powered Communication Agent

**SocialSelf** is an intelligent, multi‑platform AI assistant that automates your digital presence. It handles incoming phone calls, replies to WhatsApp messages, Instagram DMs, Telegram, Twitter, and LinkedIn – all with a human‑like voice and emotional intelligence.

Built as a 3‑hour sprint for the **GoDaddy Buildathon**, SocialSelf demonstrates rapid prototyping of Generative AI for real‑world business needs.

---

## ✨ Key Features

- 🎭 **Sentiment‑Based Tone Control** – Responds in 8 distinct personalities (polite, aggressive, humble, sentimental, friendly, professional, sarcastic, and more).
- 🧠 **Real‑Time Sentiment Analysis** – Detects the user's emotion and adapts its tone automatically.
- 💬 **Multi‑Platform Auto‑Reply** – Handles incoming messages across **6+ platforms** (WhatsApp, Instagram DMs, Telegram, Twitter, LinkedIn, and phone calls).
- 📞 **AI Voice Calls** – Inbound/outbound voice calls powered by **Twilio Programmable Voice**.
- 🔐 **User Authentication** – Secure password hashing for user accounts.
- 🧩 **Conversation Memory** – Remembers names, past topics, and user preferences across sessions.
- ⚡ **Ultra‑Fast Responses** – Powered by **Groq's LPU** technology for sub‑second latency.

---

## 🛠️ Tech Stack

| Category | Technologies |
|----------|--------------|
| **Backend** | FastAPI (Python) |
| **AI / LLM** | Groq API (Llama 3.3 70B) |
| **Voice & Messaging** | Twilio Programmable Voice & WhatsApp Sandbox |
| **Database** | SQLite (local) / Airo's built‑in PostgreSQL |
| **Deployment** | Render / GoDaddy Airo App Builder |
| **Frontend** | HTML/CSS/JS with Jinja2 templates |
